# Default Splat

Drop a Gaussian Splat file here named **`scene`** with one of these extensions:

| File | Format |
|------|--------|
| `scene.ply` | Standard PLY |
| `scene.compressed.ply` | Compressed PLY |
| `scene.sog` | SOG (super-compressed) |

The app will automatically load the first match it finds (checked in the order above).
If none exist, the app starts empty and you can load a file using the toolbar button.

---

Supported input formats: `.ply`, `.compressed.ply`, `.sog`
